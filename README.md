# E-Commerce Website (DesiCart-Learning Project)

This project is a simple e-commerce website created for learning and practicing core web development skills using HTML, CSS, and JavaScript.  
The goal of this project is to understand how an online store interface works, including product display, layout design, and basic interactivity.

## Project Purpose

This project was built to improve understanding of:
- Structuring webpages using HTML
- Styling layouts using CSS
- Adding interactivity using JavaScript
- Designing user-friendly interfaces
- Implementing basic e-commerce features

It is not intended to be a production-ready application, but rather a hands-on practice project.

## Features

- Product listing page
- Simple and clean UI design
- Responsive layout
- Basic cart functionality (if implemented)
- Navigation bar
- Product cards with images, price, and details
- Organized folder structure

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
